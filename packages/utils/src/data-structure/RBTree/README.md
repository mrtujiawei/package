# 红黑树

## 定义

- 节点是红色或黑色
- 根节点是黑色
- 所有叶子都是黑子 (叶节点下面的空节点)
- 每个红色节点的两个子节点都是黑色
- 从任一节结点其每个叶子的所有路径都包含相同数目的黑色节点

## 性质

> 从根到叶子的最长的可能路径不多于最短的可能路径的两倍长

## 平衡

### 插入

```javascript
1. 如果被插入的节点是根节点
  直接把此节点涂为黑色
2. 被插入的节点的父节点是黑色
  什么也不需要做
3. 被插入的节点的父节点是红色的
  3.1 当前节点的祖父节点的另一个子节点（叔叔节点）也是红色
    3.1.1 将父节点设为黑色
    3.1.2 将叔叔节点设为黑色
    3.1.3 将祖父节点设为红色
    3.1.4 将祖父节点设为当前节点(红色节点),之后继续对当前节点进行操作(递归)
  3.2 叔叔节点是黑色节点,且当前节点是其父节点的右孩子
    3.2.1 将父节点作为新的当前节点
    3.2.2. 以新的当前节点为支点进行左旋
  3.3 叔叔节点是黑色，且当前节点是其父节点的做孩子
    3.3.1 将父节点设为黑色
    3.3.2 将祖父节点设为红色
    3.3.3 以祖父节点为支点进行右旋
    3.3.4 右旋后，重新设置x的兄弟节点
  3.4 x的兄弟节点是黑色,x的兄弟节点的右孩子是红色的，x的兄弟节点的左孩子任意颜色
    3.4.1 将x父节点颜色，赋值给x的兄弟节点
    3.4.2 将x父节点设为 黑色
    3.4.3 将x兄弟节点的右子节设为黑色
    3.4.4 对x的父节点进行左旋
``` 
### 删除

> 删除后把原来节点的颜色叠加到新的节点上

```javascript
1. x执行一个 红+黑 节点
  将x设为一个黑节点
2. x指向根
  将x设为一个黑节点
3. 其他
  3.1 x的兄弟节点是红色
    3.1.1 将x的兄弟节点设为黑色
    3.1.2 将x的父节点设为红色
    3.1.3 将x的父节点进行左旋
    3.1.4 左旋后，重新设置x的兄弟节点
  3.2 x的兄弟节点是黑色，x的兄弟节点的两个孩子都是黑色
```
